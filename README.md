1. HomeEquityLaons
   The main objective of this project was to identify the key features that distinguish good and bad bank customers using the HMEQ dataset. The analysis includes several advanced data-analytics techniques, such as::
   - MICE imputation
   - SMOTE sample balancing
   - Ensemble learning algorithms (Gradient Boosting & Random Forest)
   - Clustering algorithms (PCA & K-means)
